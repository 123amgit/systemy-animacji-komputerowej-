# Systemy Animacji Komputerowej 

Repozytorium zawiera laboratoria z przedmiotu **Systemy Animacji Komputerowei** realizowanego na UWS w Siedlcach w semestrze letnim 2026.

## Autor
Aliaksei Makarevich

## Struktura repozytorium
Każde laboratorium znajduje się w osobnym katalogu w głównej gałęzi `main`.

Przykładowa struktura:
- `lab_01`
- `lab_02`
- `lab_03`

## Zasady organizacji
- Każde laboratorium umieszczane jest w osobnym katalogu, np. `lab_01`, `lab_02`.
- W każdym katalogu laboratorium znajduje się osobny plik `README.md` z opisem wykonania zadania.
- Nazwy plików i katalogów nie zawierają polskich znaków ani spacji.
- Praca prowadzona jest bezpośrednio w tym repozytorium, w gałęzi `main`.

## Cel repozytorium
Repozytorium służy do porządkowania, archiwizacji i prezentacji kolejnych laboratoriów wykonanych w trakcie zajęć.
